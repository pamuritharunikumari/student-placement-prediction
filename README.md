# Student Placement Prediction Using Machine Learning (Logistic Regression)

## Project Overview

This project predicts whether a student is likely to get placed based on academic, technical, and extracurricular factors.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Logistic Regression
- Joblib

## Workflow

1. Data Collection
2. Data Preprocessing
3. Label Encoding
4. Correlation Analysis
5. Train-Test Split
6. Feature Scaling
7. Logistic Regression
8. Model Evaluation
